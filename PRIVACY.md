# Privacy Policy

**Effective Date: July 4, 2026**

This Privacy Policy describes how the **Python Docs Beautifier - Material Theme** Chrome extension ("the Extension") handles user data.

## 1. No Data Collection
The Extension does not collect, store, or transmit any personally identifiable information, browsing history, or user data. It runs entirely within your browser sandboxed environment.

## 2. Local Storage of Preferences
The Extension requires the `storage` permission solely to save your local configuration preferences (such as your chosen theme, sidebar collapse state, and settings) across pages. 
* These settings are stored locally in your browser's extension storage sync/local area.
* No settings or data are ever sent to external servers or third parties.

## 3. Remote Communications
The Extension operates entirely offline and makes zero network requests. It is a pure CSS/JS injector designed to enhance the visual presentation of the official Python documentation pages.

## 4. Changes to This Policy
If we update this Extension in a way that changes data handling practices, we will update this Privacy Policy accordingly. However, our commitment to zero data collection will remain.

## 5. Contact
If you have any questions, feel free to open an issue on the project repository.
